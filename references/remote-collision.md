# Remote Collision Protocol

远距碰撞用于制造真正的新连接，而不是装饰性比喻。

## 一、选择两个节点

- A：当前内容图中的活跃节点；
- B：联想库中的远距离材料。

B 最好来自不同领域、不同尺度或不同文化语境。

## 二、强制生成三个桥梁

不要看到第一个类比就停止。至少尝试：

1. 因果机制桥梁；
2. 激励或博弈桥梁；
3. 信息、选择、反馈或进化桥梁。

## 三、机制桥梁清单

可检查：

- 信息不对称；
- 选择偏差；
- 反馈回路；
- 路径依赖；
- 激励错配；
- 信号与真实质量分离；
- 过拟合；
- 幸存者偏差；
- 网络效应；
- 规模效应；
- 权力与解释权；
- 适应与自然选择；
- 风险转移；
- 期权价值；
- 叙事与合法性；
- 资源竞争；
- 注意力稀缺。

## 四、三项验收

### 共享机制

用一句话说清 A 与 B 真正共同的机制。

### 新推论

写出连接前不明显、连接后可以推出的新判断。

### 可发布性

说明它适合形成观点、教程、案例、实验还是故事。

## 五、反空类比检查

以下连接通常不合格：

- “人生像下棋”；
- “创业像恋爱”；
- “内容像战争”；
- 只靠相同词汇，没有相同机制；
- 不能产生可检验的新判断。
