# Content Review Diagnostics

这些是诊断启发，不是平台秘密算法。优先和创作者自己的可比历史中位数比较。

## 1. 分发层

现象：展示或播放显著低于基线。

可能原因：样本仍早、发布时间、题材过窄、平台波动、账号分发不稳定。

不要立刻杀死母题。

## 2. 包装层

现象：有展示，但点击、进入或前几秒留存低。

检查：标题是否具体、封面是否易读、受众是否一眼认出、开头是否立刻兑现。

下一次只改标题、封面或开头之一。

## 3. 消费层

现象：点击不差，但完播、停留或阅读深度低。

可能原因：铺垫过长、承诺与正文不一致、信息密度不足、结构混乱。

## 4. 价值层

现象：看完表现尚可，但收藏、分享和深度评论弱。

可能原因：内容可看但不可复用、没有独特机制、观点过安全、行动价值不足。

## 5. 转化层

现象：有互动，但关注、主页访问、私信、线索或成交弱。

可能原因：账号长期价值不清、没有系列预期、吸引了错误人群、行动引导不自然。

## 常见异常

- 高点击 + 低完播：包装强于兑现；
- 低点击 + 高完播：先改包装，不要杀母题；
- 高收藏 + 低关注：单篇工具价值强，账号长期承诺弱；
- 高评论 + 低收藏：争议或情绪强，不一定能沉淀知识支柱；
- 高播放 + 低意图：泛流量，不一定服务北极星；
- 普通数据 + 深度评论：可能存在窄而高价值的子人群；
- 评论持续求后续：优先形成 CHILD；
- 失败帖中某个段落被反复引用：提取局部成功元素作为新种子。
